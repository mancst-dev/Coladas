## Application: Extract data from .pdf to .xlsx

The app has been developed to help consultants within a customs agency extract information from PDF files, saving them a great deal of time.

### Features:
- Boostrap 5.
- CSS.
- JavaScript.
- Python 3.12.0

### Local Development

### Link download Python  3.12.0 :
[Python 3.12.0](https://www.python.org/downloads/release/python-3120/)

#### Once Python has been installed, create the environment variable:
- Variable name:
```
Path
```
- Locate value variable:
```
C:\Users\You User\AppData\Local\Programs\Python\Python312\Scripts\
```

#### Library list:
- camelot-py
- flask
- openpyxl
- pandas
- streamlit

To install the libraries, list the packages after ``pip install``, separating them with a space, using:

```
pip install pandas camelot-py openpyxl streamlit flask
```

#### How to use the software:

Clone the repository and open the project in your favourite code editor.

Run the `app.py` file. Flask will start up and launch a local server at `http://127.0.0.1:5000/`.

**WARNING: This is a development server. Do not use it in a production deployment. Use a production WSGI server instead.**

You'll finally be able to use the exhaust fan.
